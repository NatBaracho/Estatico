
# ⚡ Métodos Estáticos em C#

Este repositório contém um projeto prático de estudo em **C#** focado em demonstrar o uso de **Métodos Estáticos** na Programação Orientada a Objetos (POO).

## 🚀 Sobre o Projeto
O código é um exemplo direto e simples de uma calculadora de soma. Ele demonstra como podemos criar funções utilitárias que pertencem à classe em si, e não aos objetos gerados a partir dela.

## 🧠 Conceitos de POO Aplicados
Este projeto explora os seguintes conceitos:

* **Modificador `static`:** Aplicado ao método `somar`, indicando que ele é um método estático.
* **Acesso direto via Classe:** Como o método é estático, ele é chamado diretamente pelo nome da classe (`Exemplo.somar()`), eliminando a necessidade de instanciar um objeto na memória usando a palavra reservada `new`.
* **Organização de código:** Ideal para funções globais ou utilitárias (como cálculos matemáticos) que não dependem do estado (atributos) de um objeto específico.
